### 3. Технические детали
- Найти товары по `slug`/`name` в массиве `PRODUCTS`.
- `description: string[]` — заменить целиком на новый массив абзацев.
- `features: string[]` — заменить целиком на новый список (буллеты `•` уже отображаются иконкой Check, поэтому символ `•` из исходного текста убираем).
- Для моделей с PRO-сравнением добавить в конец `features` строку-заголовок `"Преимущества версии PRO:"` и затем PRO-пункты — рендер всё равно покажет их единым списком; альтернативно — добавить эти пункты как отдельные параграфы в `description`. **Вариант по умолчанию: PRO-сравнение идёт отдельным абзацем в конце `description`, чтобы визуально отделить.**
